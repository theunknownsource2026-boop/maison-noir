004 / Process

Built, not manufactured.

01

Source

Dead-stock fabrics and single-origin mills. We never produce new textiles when existing ones will do.

02

Pattern

Each pattern is draped on the form, not drafted on screen. The fabric decides the silhouette.

03

Construct

Hand-finished in a single atelier. One maker sees each piece from cut to final press.

04

Release

No pre-orders. No waitlists. When it's ready, it appears. First come, only come.

Stay in the dark.

Private list. No spam. First access to new pieces.
